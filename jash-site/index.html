InstallationУстановка

Clone the repository and build with Go.

Склонируйте репозиторий и соберите через Go.

terminal 
git clone https://github.com/qwantuum/jash.git
cd jash
go build ./cmd/jash
./jash

SyntaxСинтаксис

Familiar C-style syntax with dynamic types and built-in data structures.

Знакомый C-подобный синтаксис с динамической типизацией и встроенными структурами данных.

syntax.jash 
# Variables and types
name = "jash"
pi   = 3.14
nums = [1, 2, 3]
obj  = {"key": "value"}

# Control flow
if len(nums) > 0 {
    print nums[0]
}

# Functions
fn add(a, b) {
    return a + b
}

# Loops
for i in [1, 2, 3] {
    print i
}

# Lambda
result = map(arr, fn(x) { return x * 2 })
